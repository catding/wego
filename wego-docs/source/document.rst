WEGO
================================

Wego 能协助你在微信开发中专注业务逻辑摆脱微信接口调试的烦恼。

看看 WEGO 到底有多方便: ::

    # django
    @w.login_required
    def index(request):
        hello = 'Hello %s!' % request.wx_user.nickname
        return HttpResponse(hello)

    # tornado
    class IndexHandler(tornado.web.RequestHandler):
        @w.login_required
        def get(self):
            hello = 'Hello %s!' % self.wx_user.nickname
            return self.write(hello)
